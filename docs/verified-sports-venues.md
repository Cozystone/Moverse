# Moverse 검증 스포츠 시설

기준일: 2026-07-19

## 제품 적용 규칙

- 농구와 축구 이벤트는 `facility` 정보가 있고 해당 종목을 지원하는 검증 스팟에서만 생성한다.
- 예약형 시설은 공식 예약 페이지를 열어 실제 날짜와 시간을 예약하고, `구장 예약 완료`를 확인해야 이벤트를 게시할 수 있다.
- 무료 개방 코트도 독점 이용이나 대규모 행사를 보장하지 않는다.
- 지도 카드에는 이용 방식과 공식 정보 링크를 함께 표시한다.
- 시설 상태를 마지막으로 확인한 날짜를 데이터에 저장하고 정기적으로 갱신한다.
